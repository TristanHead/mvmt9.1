# Mvmt9.1
Work in progress momentum focused platformer with intent to make movement an art

Mvmt9.1 (not an abbreviation) is currently just a test, but is designed around the goal of making movement itself the fun part of the game. Currently, it is only worked on by me, but maybe eventually I'll get more people. The project is being made in Godot, and is primarily exported for web so I can test it on anything, but future plans for native binaries do exist. Here's my current plans for the game:

EVENTUALLY:  Add enemies for the player to fight
NEVER:  Make grappling less cracked
